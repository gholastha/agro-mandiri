Buat Admin Dashboard Core untuk e-commerce agrikultur dengan spesifikasi berikut:

FITUR UTAMA:
1. Layout & Komponen:
   - Sidebar responsif dengan navigasi terstruktur (Produk, Pesanan, Pelanggan, Inventaris, Pengaturan)
   - Header dengan search bar, notifikasi, dan user profile dropdown
   - Tema gelap/terang yang dapat dikonfigurasi
   - Dashboard overview dengan KPI cards (penjualan hari ini, pesanan baru, stok rendah)

2. Manajemen Produk:
   - Form CRUD produk lengkap dengan validasi React Hook Form + Zod
   - Struktur kategori untuk Pupuk, Pestisida, Benih, dan Peralatan
   - Editor atribut dinamis sesuai kategori (dosis untuk pupuk, varietas untuk benih)
   - Uploader gambar multi-file dengan preview dan crop (gunakan Supabase Storage)
   - Manajemen variasi produk (ukuran kemasan, harga grosir/eceran)

3. Inventaris & Stok:
   - Dashboard stok dengan filter dan sorting
   - Form update stok (single dan bulk)
   - Alert sistem untuk stok rendah
   - Log aktivitas inventaris dengan timestamp

4. Statistik & Grafik Dasar:
   - Chart penjualan mingguan/bulanan menggunakan Chart.js
   - Top products by revenue/quantity
   - Visualisasi performa kategori produk
   - Export data ke CSV/Excel

IMPLEMENTASI TEKNIS:
- Gunakan React dengan Tailwind CSS dan Shadcn UI untuk komponennya
- State management dengan Zustand, struktur dengan atomic design pattern
- Integrasi Supabase untuk database, autentikasi, dan storage
- Responsive design dengan mobile-first approach
- Implementasikan RLS (Row Level Security) untuk keamanan data

Output yang diharapkan adalah kode lengkap untuk admin dashboard yang modular dan dapat dikembangkan lebih lanjut. Pastikan semua validasi form berfungsi, UI responsif, dan integrasi Supabase sudah dikonfigurasi dengan benar.

SISTEM AUTENTIKASI:
1. Implementasi Auth Supabase dengan 3 metode login:

     * Form input nomor telepon dengan validasi format Indonesia
     * Sistem verifikasi OTP dengan countdown timer
     * Kode implementasi lengkap untuk signInWithOtp dan verifyOtp
   
   - Social Login (Google & Facebook):
     * Konfigurasi OAuth providers di Supabase
     * UI button dan handler untuk social sign-in
     * Proses onboarding untuk user baru dari social login

2. User Management:
   - Profil pengguna dengan informasi personal dan alamat pengiriman
   - Role-based access control (Pembeli, Admin)
   - Halaman riwayat pesanan pengguna
   - Form edit profil dan reset password

ALUR TRANSAKSI:
1. Keranjang Belanja:
   - Fungsi add to cart dengan state persistence (localStorage + Supabase)
   - UI keranjang dengan update kuantitas dan remove item
   - Kalkulasi subtotal, diskon, dan total
   - Mini cart di header dengan jumlah item dan total

2. Proses Checkout:
   - Form multi-step dengan wizard UI:
     * Step 1: Review keranjang
     * Step 2: Alamat pengiriman (dengan validasi)
     * Step 3: Metode pengiriman (integrasi RajaOngkir)
     * Step 4: Metode pembayaran (integrasi Midtrans)
     * Step 5: Konfirmasi pesanan
   - Order summary yang update real-time
   - Handling diskon dan kode promo

3. Order Management:
   - Dashboard pesanan dengan filter status
   - Detail pesanan dengan timeline status
   - Notifikasi email dan in-app untuk update status
   - Invoice generation dan download

IMPLEMENTASI TEKNIS:
- Gunakan React dengan Tailwind CSS dan Shadcn UI
- Form validation dengan React Hook Form + Zod
- State management dengan Zustand untuk cart dan checkout
- Integrasi Supabase untuk auth dan database
- Responsive design optimal untuk mobile (petani Indonesia)

Output yang diharapkan adalah sistem autentikasi yang aman dan alur transaksi yang intuitif, dengan fokus khusus pada UX yang sederhana namun powerful untuk pengguna pedesaan dengan koneksi internet yang mungkin terbatas.
