Objektif
Kembangkan aplikasi e-commerce modern untuk produk agrikultur (Pupuk, Pestisida, Benih, Peralatan) yang memudahkan petani Indonesia menemukan produk pertanian tepat guna dan murah, sekaligus meningkatkan brand awareness dan literasi agroteknologi, dengan inspirasi dari situs seperti bighaat.com. Aplikasi terdiri dari Client-side dan Admin Dashboard, dibangun menggunakan React, terintegrasi dengan Supabase untuk database, autentikasi, dan penyimpanan file. UI/UX dirancang dengan Tailwind CSS, Shadcn UI, dan komponen modular yang efisien serta responsif.
Sistem harus mudah diatur dan dikembangkan, dengan dashboard yang mengontrol komponen client-side secara real-time, termasuk tema, layout, navigasi (mega menu), dan konten.
I. Klasifikasi Fitur (Berdasarkan Prioritas & Best Practice)
1. Manajemen
Fitur operasional toko dan kontrol front-end:

Produk: Implementasikan CRUD untuk produk, termasuk variasi (misalnya, jenis pupuk, merek), harga satuan/perkuantitas, dan stok.
Kategori & Atribut:
Kategori: Pupuk (organik, anorganik), Pestisida (insektisida, herbisida), Benih (padi, jagung), Peralatan (traktor, pompa).
Atribut: Warna, ukuran kemasan, merek, dosis penggunaan.

Inventaris: Sistem manajemen stok dengan notifikasi stok rendah.
Pesanan: Kelola status pesanan (tertunda, diproses, dikirim), dukung aksi massal, dan pelacakan.
Pengiriman: Opsi pengiriman, hitung ongkos kirim, integrasikan pelacakan.
Pelanggan: Simpan detail pelanggan, histori pembelian, preferensi, segmentasi.
Menu Navigasi (Mega Menu): Konfigurasi dari dashboard, update real-time di client-side.
SEO: Kelola judul, deskripsi meta, slug, dan tag SEO untuk produk dan halaman.

2. Pengaturan
Pengaturan sistem dan branding via dashboard:

Profil Perusahaan & Info Toko: Nama toko, logo, alamat, jam operasional.
Tema dan Tampilan: Kontrol warna, logo, layout, font dari dashboard.
Akun & Role: Kelola akun pengguna/admin dengan RBAC.
Notifikasi: Email, toast alert, push notification untuk pesanan, stok, dll.
Integrasi Database: Supabase untuk data real-time dan penyimpanan.
Integrasi Eksternal: Email marketing (Mailchimp), pembayaran (Midtrans, Stripe), pengiriman (RajaOngkir).

3. Laporan & Analitik
Fitur untuk pengambilan keputusan berbasis data:

Laporan Penjualan: Berdasarkan periode, kategori, produk, pelanggan.
Laporan Inventaris: Produk dengan rotasi cepat, stok menipis.
Analisis Pelanggan: Pelanggan setia, retensi, perilaku pembelian.
Laporan Pemasaran: Performa kampanye, konversi, promosi.
Ekspor Data: Format CSV, Excel, JSON.

4. Kontrol & Kustomisasi Client-side
Komponen tampilan front-end dikontrol via dashboard:

Mega Menu: Editor drag-and-drop untuk struktur menu navigasi.
Tampilan Halaman Utama: Pilih layout, urutan section (hero, produk unggulan, promo).
Produk Unggulan & Promosi: Kurasi dari dashboard.
Banner & Konten Marketing: Upload dan atur banner/promosi.
Tema Dinamis: Ubah warna, font, mode gelap/terang tanpa deploy ulang.
Literasi Agroteknologi: Bagian untuk artikel, tips, dan panduan pertanian.

5. Keamanan & Logging
Menjamin integritas dan keamanan data:

Autentikasi: Gunakan Supabase Auth dengan login via:
Telepon: OTP dikirim melalui SMS.
Google: Login sosial melalui Google.
Facebook: Login sosial melalui Facebook.

Role-based Access Control (RBAC): Kontrol akses granular dengan RLS.
Audit Log: Catat riwayat aktivitas admin dan pengguna.
Proteksi Data: Validasi input, rate limiting, enkripsi data sensitif.

II. Praktik Teknis yang Harus Dipatuhi

Stack Frontend: React (Next.js opsional untuk SEO), Tailwind CSS, Shadcn UI, Zustand/Redux untuk state management.
Backend & DB: Supabase (PostgreSQL, Auth, Storage, Realtime).
Visualisasi Data: Chart.js, Recharts, atau Nivo.
Routing: React Router v6 atau App Router (Next.js).
Form Handling: React Hook Form dengan Zod/Yup untuk validasi.
Responsiveness: Desain mobile-first, layout berbasis grid, dukungan mode gelap.
Modularisasi Komponen: Pola desain atomik untuk UI.
Internationalization (Opsional): react-i18next untuk multi-bahasa.

III. Pengembangan Lanjutan

Progressive Web App (PWA): Dukung akses offline dan loading cepat.
Drag-and-drop Builder: Kustomisasi homepage dan landing page.
Mini CMS: Kelola konten blog/promo.
AI-assisted Recommendations: Rekomendasi produk berdasarkan perilaku pelanggan.
Multi-vendor Support: Ekspansi ke model marketplace.

IV. Implementasi Spesifikasi Khusus
Kategori Produk

Pupuk: Organik, anorganik, mikroba; atribut seperti merek, dosis, kemasan.
Pestisida: Insektisida, herbisida, fungisida; atribut seperti bahan aktif, target hama.
Benih: Padi, jagung, sayuran; atribut seperti varietas, masa tanam.
Peralatan: Traktor, pompa, alat semprot; atribut seperti daya, kapasitas.
Implementasikan filter dan pencarian untuk memudahkan petani menemukan produk.

Fitur untuk Petani Indonesia

Pencarian dan Filter: Cari berdasarkan kategori, harga, lokasi pengiriman.
Literasi Agroteknologi: Sediakan artikel dan panduan (misalnya, cara aplikasi pupuk, pemilihan benih).
Desain Mobile-first: Antarmuka sederhana, dioptimalkan untuk smartphone.
Bahasa Lokal: Gunakan Bahasa Indonesia, dengan opsi Bahasa Inggris.

Inspirasi dari bighaat.com

Struktur Kategori: Tiru organisasi kategori yang jelas dan filter produk.

Konfigurasi Autentikasi

Phone Authentication:
Aktifkan di dashboard Supabase pada Auth Providers.
Konfigurasikan provider SMS (Twilio, Vonage, MessageBird).
Implementasikan login OTP:const { data, error } = await supabase.auth.signInWithOtp({
  phone: '+628123456789',
  options: { shouldCreateUser: true },
});
const { data, error } = await supabase.auth.verifyOtp({
  phone: '+628123456789',
  token: '123456',
  type: 'sms',
});


Google dan Facebook Login:
Aktifkan Google dan Facebook di Auth Providers.
Dapatkan kredensial dari Google Developer Console dan Facebook Developer.
Implementasikan:const { data, error } = await supabase.auth.signInWithOAuth({ provider: 'google' });
const { data, error } = await supabase.auth.signInWithOAuth({ provider: 'facebook' });


V. Catatan Tambahan

Supabase Real-time: Gunakan untuk pembaruan stok dan status pesanan.
Next.js: Pertimbangkan untuk SEO dan performa.
Keamanan: Pastikan autentikasi menggunakan JWT, dengan RBAC, audit log, dan proteksi data.
Aksesibilitas: Optimalkan untuk pengguna mobile di daerah rural Indonesia.
